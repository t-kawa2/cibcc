cibcc (chibicc multi-language implementation)
This project is a personal learning journey inspired by chibicc.
I named it cibcc to distinguish it from the original and to focus on my own implementation across multiple languages.
🚀 Supported Languages
You can find the implementation for each language in the following directories:

    /Go
    /C++
    /Java
    /Python
    /Rust

✨ Features

    Step-by-Step: Each directory contains code organized by steps (Step 0 and beyond).
    Ready to Run: Each step includes its own Makefile and test.sh.
    Learning Resource: Based on the Japanese guide "野良プログラマーのCS日記 (Nora Programmer's CS Diary)".

🛠 How to Use

    Enter the directory of the language you want to try.
    Go to the specific step directory.
    Run the following commands:

bash

make
make test
make clean
