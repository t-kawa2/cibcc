This project demonstrates a program that returns integral functions using yacc (bison) and lex (flex).

    Step 1:
    Run the program with ./int1. If you input a constant like 3, it returns 3x + c.
    Step 2:
    Supports polynomials. For example, inputting 3x^2+2x-1 returns x^3+x^2-x+c. (Note: ^ represents exponentiation).
    Step 3:
    Supports exponential functions. For example, inputting xe^x returns xe^x - e^x + c. It also handles complex expressions like x^2e^x.
